# PoryDex

Welcome to the PoryDex, a tool made to help play Pokemon Odyssey! Just hop on over to <a href="https://porydex.app">the PoryDex site!</a>

# Features

- Compendium of Moves, Abilities, Roles, and Pokemon!

# Setup

- Install nodejs + npm
- run `npm install`
- run `npm start`
